Research Interest

 I work in the field of Android security analysis and source code vulnerability analysis. Currently, I focus on the following research topics:
  • Android Malware Analysis: Android Malware Detection and Classification
  • Vulnerable Code Clone Analysis: Code Clone Detection and Vulnerable Code Clone Discovery
  • Intelligent Vulnerability Analysis: Enhancing the Accuracy, Robustness, and Interpretability of Deep Leaning-based Vulnerability Detection
