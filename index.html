Research Interest

 Currently, I focus on the following research topics:
  • Mobile Security: Android Malware Detection, Classification, and Interpretation
  • Open Source Security: Code Clone Detection, Vulnerable Code Clone Discovery, and Software Composition Analysis
  • AI Security: Enhancing the Accuracy, Robustness, and Interpretability of Deep Leaning-based Vulnerability Detection
  • Blockchain Security: Smart Contract Clone Detection and Vulnerability Detection
  • Web Security: JavaScript Malware Detection and Robustness Analysis
